# Sauce Demo – Test Scenarios

## 1. Login checklist

- [ ] Verify login with valid credentials
- [ ] Verify login with invalid username
- [ ] Verify login with invalid password
- [ ] Verify login with empty username field
- [ ] Verify login with empty password field
- [ ] Verify login with both fields empty
- [ ] Verify login with locked out user
- [ ] Verify error message after failed login
- [ ] Verify error message can be closed
- [ ] Verify logout functionality

## 2. Products Page checklist

- [ ] Verify products page loads after successful login
- [ ] Verify product names are displayed correctly
- [ ] Verify product prices are displayed correctly
- [ ] Verify product images are displayed correctly
- [ ] Verify product details page opens correctly
- [ ] Verify user can return from product details to products list
- [ ] Verify sorting by name A to Z
- [ ] Verify sorting by name Z to A
- [ ] Verify sorting by price low to high
- [ ] Verify sorting by price high to low

## 3. Cart checklist

- [ ] Verify user can add one product to cart
- [ ] Verify cart badge updates after adding product
- [ ] Verify user can add multiple products to cart
- [ ] Verify user can remove product from products page
- [ ] Verify user can remove product from cart page
- [ ] Verify cart page displays selected products correctly
- [ ] Verify product name and price in cart match products page
- [ ] Verify user can continue shopping from cart
- [ ] Verify empty cart behavior
- [ ] Verify cart content remains after page refresh

## 4. Checkout checklist

- [ ] Verify checkout can be started from cart
- [ ] Verify checkout form is displayed correctly
- [ ] Verify checkout with valid first name, last name and postal code
- [ ] Verify checkout with empty first name
- [ ] Verify checkout with empty last name
- [ ] Verify checkout with empty postal code
- [ ] Verify checkout error messages are displayed correctly
- [ ] Verify checkout can be cancelled
- [ ] Verify checkout overview displays correct products
- [ ] Verify item total is calculated correctly
- [ ] Verify tax is displayed
- [ ] Verify final total is calculated correctly
- [ ] Verify order can be completed successfully
- [ ] Verify confirmation message after successful order

## 5. Navigation & Session checklist

- [ ] Verify side menu opens correctly
- [ ] Verify side menu closes correctly
- [ ] Verify “All Items” link redirects to products page
- [ ] Verify “About” link redirects correctly
- [ ] Verify logout redirects user to login page
- [ ] Verify user cannot access products page after logout using browser back button

## 6. Usability checklist

- [ ] Verify main elements are aligned correctly on desktop
- [ ] Verify buttons are visible and readable
- [ ] Verify error messages are clear and understandable
- [ ] Verify form fields are labeled correctly
- [ ] Verify layout remains usable on smaller browser width
- [ ] Verify important actions are easy to identify
